## Introduction

A Neovim wrapper for Rust, allowing you to embed the text editor into a program and use it as a library.

## Build Instructions

### Linux (apt-get)

```Shell
apt-get install libtool autoconf automake cmake libncurses5-dev g++ pkg-config unzip
cargo build
```

### Linux (yum)

```Shell
yum install autoconf automake cmake gcc gcc-c++ libtool ncurses-devel pkgconfig
cargo build
```

### OS X (homebrew)

```Shell
brew install libtool automake cmake pkg-config
cargo build
```
